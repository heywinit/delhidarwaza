# Delhi Darwaza

A simple trading exchange made in Odin

## Why?

- I wanted to try out Odin
- I like exchanges
- I like Odin & exchanges

## Building

To build the project:

```bash
odin build src
```

To run:

```bash
odin run src
```

## Requirements

- Odin compiler (https://odin-lang.org/) (duh)
